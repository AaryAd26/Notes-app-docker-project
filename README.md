# 🚀 Notes App (Dockerized)

## 📌 What This App Does

A simple Notes Web Application built with Node.js and Express.
Users can add and view notes via a clean UI.

---

## 🐳 Run with Docker Compose

```bash
docker compose up --build
```

---

## 🌐 Access

Open browser:
http://localhost:5000

---

## ⚙️ Environment Variables

| Variable  | Description               |
| --------- | ------------------------- |
| PORT      | App port                  |
| MONGO_URL | MongoDB connection string |

---

## 🧱 Services

* Node.js App
* MongoDB Database
* Volume for persistence
* Custom Docker network
